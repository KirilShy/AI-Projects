# AI-Projects

A consolidated collection of AI/ML projects, each in its own subdirectory.

## Projects

### [brain-tumor-mri-classifier](brain-tumor-mri-classifier/)
PyTorch CNN for detecting brain tumors from MRI scans. Trains and evaluates end-to-end in a Jupyter notebook with GPU (CUDA) support, achieving 96.57% test accuracy on 128×128 images.

### [flower-image-classification](flower-image-classification/)
CNN-based image classifier built with TensorFlow/Keras that identifies flower species from photos.

### [mnist-cnn-keras](mnist-cnn-keras/)
Keras/TensorFlow CNN for handwritten digit recognition on MNIST, achieving ~98.28% test accuracy with only a few minutes of training.

### [ai-snake-rl](ai-snake-rl/)
A Snake-playing agent trained with reinforcement learning, with a Pygame environment and a modular agent/model/training loop.

### [john-assistant](john-assistant/)
A modular personal AI assistant built from scratch, supporting multi-turn chat, a GUI, voice input (Whisper), and natural text-to-speech (ElevenLabs).
